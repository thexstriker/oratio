# Oratio.ai

This is Oratio.ai, a machine learning model that was built to detect common phonological disorders based on changes in phoneme transcriptions. A 563 word database that have phonemes vulnerable to common phonological disorders was created to train the model.

## Tech Stack

**Languages & Frameworks:** Python, TensorFlow
